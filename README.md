# 🩺 Diabetic Retinopathy Screening System

## 📌 Overview
A Python-based deep learning system designed to **detect diabetic retinopathy from retinal fundus images**.  
The project analyzes retinal images and identifies patterns related to **damage in blood vessels caused by diabetes**, helping in early screening of eye disease.


## 🔬 Processing Pipeline

```
Retinal Image → Preprocessing → CNN Model → DR / No DR Prediction
```


## 🚀 Features

- Upload retinal fundus images for analysis  
- Image preprocessing and normalization  
- Deep learning based classification  
- Detection of **Diabetic Retinopathy presence**  
- Visualization of prediction results  
- Automated screening assistance  


## 🛠 Technologies Used

- Python  
- TensorFlow / Keras  
- OpenCV  
- NumPy  
- Matplotlib  
- Pandas  


## 🎯 Applications

- Early screening of **Diabetic Retinopathy**
- Medical image analysis research
- AI in healthcare projects
- Biomedical engineering studies
- Computer vision learning projects

